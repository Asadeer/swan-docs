**解释**：给定的 (x, y) 位置绘制文本描边的方法。

**方法参数**：String text, Number x, Number y, Number maxWidth

**`text`参数说明**：要绘制的文本

**`x`参数说明**：文本起始点的 x 轴坐标

**`y`参数说明**：文本起始点的 y 轴坐标

**`maxWidth`参数说明**：需要绘制的最大宽度，可选。

**代码示例**：

```js
canvasContext.strokeText(text, x, y, maxWidth);
```
